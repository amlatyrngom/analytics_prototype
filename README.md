# Smart IDs
Repo for Smart IDs.

## Current Features
* Table Storage.
* Loading tables (including TPCH tables) from csv.
* Simple Vectorized Query Engine: Scan, Filter, Projection, Joins, Aggregations.

## Next Features
* Query Engine Features:
  * Parallel Execution.
  * Indexes.
  * NULL support.
* Parser.
* Optimizer & Cost Model.
